# 前端常见bug

获取

- 查询条件变更，但是前端没改
- 查询参数漏传
  - 比如需要不分页，没传isPaging为false，导致只拿到了第一页的数据

**通用bug**

- 类型不一致：之前只接收字符串，现在传递的却是数组
- 没有兼容多类型：比如需要字符串，也需要兼容数组
- 类型判断错误：比如通过.length来判断是否为数组，但是传递的是字符串也能通过这个判断，导致出错
- 未考虑边界情况
  - 传递的是空数组，但是没有判断，导致出错
  - 读到边界外去了
