**表设计技巧**

- 链路不要太长，不然调试、定位问题很困难
  - 比如角色表》权限表》权限子表，权限表和权限子表的外键都应该是角色表id，而不是权限子表去

数据库可以设计子表，但是一般不要超过三层，不然出了问题，很难定位
